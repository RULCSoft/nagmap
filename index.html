Nagmap

NagMap - standalone integration of Nagios or Icinga into Google Maps

Download this project as a .zip file Download this project as a tar.gz file

Intro

NagMap project is an initiative to create super-simple application to integrate Nagios or Icinga with Google maps. The integration aims to visualize current status of network devices on aerial photography images.

The single point of contact for this project is here: http://exchange.nagios.org/directory/Addons/Maps-and-Diagrams/nagmap/details - please use this site for contacts and more information.
